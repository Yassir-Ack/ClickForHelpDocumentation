Click for help cfh

Let's talk about click for help. Take the notes in terminal terminote

        
        terminote ~ $

Installation

  • npm
  • yarn
  • cdn
            $  npm install cfh
            $  yarn add cfh
            $  https://clickforhelp.js.org

Fast & Light

Pure & optimized javascript file using ES6, well-structured! Readable code with comments. Less than one hundred line.

Easy to Use

Just put the script or CDN after body tag and that's all. You get it.

Open Source

Licened under MIT and hosted on GitHub.

Configuration

Maintain all your settings in a single setting.json file. Never need to redo the setting every single time jotting down a note.

Two Ways to Use

You can use it with setting.json file to configure your params, or just put your params as attribute inside cfh script!

Secure

Configure your params locally. No one can get to your secrets!
